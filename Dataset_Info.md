## Data Source
The dataset used in this project is a publicly available Blinkit sales dataset obtained for analytical and educational purposes.

> Source: Kaggle (Blinkit Sales Dataset)

---

## Dataset Size
- Number of rows: ~8,500 records
- Number of columns: 12

---

## Key Columns
| Column Name        | Description                                   |
|--------------------|-----------------------------------------------|
| Item Identifier    | Unique identifier for each product            |
| Item Type          | Category of the product                       |
| Item Fat Content   | Indicates whether item is low fat or regular  |
| Item Visibility    | Percentage visibility of product              |
| Item Weight        | Weight of the item                            |
| Outlet Identifier  | Unique ID for each outlet                     |
| Outlet Size        | Size category of outlet (Small/Medium/Large)  |
| Outlet Location    | Tier of city (Tier 1, Tier 2, Tier 3)          |
| Outlet Type        | Type of outlet                                |
| Sales              | Total sales value                             |
| Rating             | Customer rating                               |
